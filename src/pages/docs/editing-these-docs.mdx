# Editing These Docs

If you want to submit a request to edit these docs, do the following.

1. [Create a GitHub account](https://github.com/signup) if you don't have one.
2. Click the `Edit this page` button on the right side of the screen.
   - Alternatively, visit the [GitHub repository](https://github.com/WiiGolfQ/website/tree/main/src/pages/docs) and find the file associated with the page to edit.
   - You can also add pages by clicking the `Add file` button. See [this page of the Nextra documentation](https://nextra.site/docs/guide/organize-files) (Nextra is the platform these docs are built on) which describes the correct file structure.
3. Click the `Edit this file` button.
   - Fork the repository if you haven't already.

![Edit this file button on GitHub](https://i.ibb.co/7XG3LDy/editing-step-3.jpg)

4. Make your changes and click `Commit changes...`.

![Commit changes button on GitHub](https://i.ibb.co/QdT8WzL/image.png)

5. Click `Create pull request`.

![Create pull request button on GitHub](https://i.ibb.co/mtnGjbk/image.png)

6. Write a basic title that describes what you changed.
7. Click `Create pull request` again.

![Creating and submitting a pull request on GitHub](https://i.ibb.co/PtC51sF/image.png)

With that, you have successfully submitted a request to edit these docs.
